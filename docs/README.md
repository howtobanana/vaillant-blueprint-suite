# Vaillant Blueprint Suite

Dieses Repository enthält die Vaillant Automations-Suite für Home Assistant.

## Aktive Module
- `blueprints/vaillant_heizungs-steuerung.yaml` (Core/Backbone)
- `blueprints/vaillant_geofencing.yaml`
- `blueprints/vaillant_lüftungs-steuerung.yaml`
- `blueprints/vaillant_system-diagnose.yaml`

## Zusatzmodul
- `blueprints/vaillant_notification_tester.yaml`

## Struktur
- `blueprints/` aktive produktive Blueprints
- `releases/legacy/` alte Snapshots und Zwischenstände
- `docs/` Doku, Changelog, Kompatibilität

## Namensregel
- Dateinamen sind deutsch und stabil.
- Versionierung erfolgt über Git-Tags und Blueprint-Header, nicht über Dateinamen.
